縮略碼表

#输入法

縮略碼表 总表

  • 英文字母 = a-z ＆ zm
  • 計數字符 = 0-9 ＆ sz
  • 查看使用說明 = ? + 縮略碼
  • 按相關字母檢索 = 編碼 + 字母
  • 按數字關係檢索 = 編碼 + 數字
  • 切換至半角形態 = 編碼 + bj
  • 切換至大寫形態 = 編碼 + dx
  • 切換至小寫形態 = 編碼 + xx
  • 特殊符號 = ts
  • 標點符號 = bd
  • 漢語拼音 = py
  • 註音符號 = zy
  • 漢字筆畫 = bh
  • 偏旁部首 = bs/pp ＆ kx
  • 漢字結構 = jg
  • 日文假名 = hg / kk / rw
  • 希臘字母 = xl
  • 俄語字母 = ey
  • 希伯來文 = xblw / xw
  • 拉丁字母 = ld
  • 國際音標 = yb
  • 羅馬數字 = lm
  • 計量單位 = dw
  • 貨币單位 = hb
  • 數學算符 = sx
  • 分數符號 = fs
  • 聲調調型 = dx / sd
  • 時間日期 = sj
  • 易經八卦 = yj ＆ bg
  • 太玄經爻 = txj
  • 盲文點字 = mw
  • 漢文批註 = pz
  • 上標下標 = sb ＆ xb
  • 星號標記 = xh
  • 指向箭頭 = jt
  • 製表符號 = zb
  • 組合方塊 = fk
  • 幾何圖形 = tx
  • 電腦界面 = dn
  • 信號標誌 = bz
  • 人物表情 = bq
  • 兩性關係 = lx
  • 音樂樂譜 = yy
  • 天氣狀況 = tq
  • 黃道星宮 = xz
  • 宇宙天體 = tt
  • 煉金符號 = lj
  • 遊戲紙牌 = qp ＆ gp ＆ mj ＆ pk ＆ tl
  • 棋子骰子 = qp ＆ tz ＆ xq
  • 國家地區 = dq
  • 空白字符 = kb / kg
  • 挖掘彩蛋 = 隨意組合

縮略碼表 明细表

bd ─╴標點符號╶─────

  • bd 半角 = bd + bj
  • bd 全角 = bd + qj
  • bd 日文 = bd + rw
  • bd 小型 = bd + xx
  • bd 大型 = bd + dx
  • bd 直書 = bd + zs
  • bd 橫槓 = bd + hg
  • bd 上標 = bd + sb
  • bd 引號 = bd + yh
  • bd 句點 = bd + d

    bh ─╴漢字筆畫╶─────

  • bh 按名稱 = bh + 簡拼
  • bh 按折點數 = bh + 數字

    jg ─╴字形結構╶─────

  • jg 按名稱 = jg + 簡拼
  • jg 簡拼有 zy/sx/zs/zx/ys/hh/b/w/…

    lm ─╴羅馬數字╶─────

  • lm 古代記法 = lm + old

    sz ─╴其它數字╶─────

  • sz 算籌 = sz + sc
  • sz 蘇州碼 = sz + sz
  • sz 正字計數 = sz + zz
  • sz 西方統計 = sz + xf

    sx ─╴數學算符╶─────

  • sx 常數 = sx + cs
  • sx 集合 = sx + jh
  • sx 邏輯 = sx + lj
  • sx 省略 = sx + sl
  • sx 其它 = sx + 簡拼

    xl ─╴希臘字母╶─────

  • xl 按排序 = xl + 數字
  • xl 按名稱 = xl + 英文名

    rw ─╴日文假名╶─────

  • rw 按日本式 = rw + 羅馬字
  • rw 按黑本式 = rw + 羅馬字
  • rw 外語轉寫 = rw + wy

    hg ─╴平假名╶────

  • hg 按日本式 = hg + 羅馬字
  • hg 按黑本式 = hg + 羅馬字

    kk ─╴片假名╶────

  • kk 按日本式 = kk + 羅馬字
  • kk 按黑本式 = kk + 羅馬字
  • kk 方塊組合 = kk + fk

    dx ─╴聲調調型╶─────

  • dx 五度標記 = dx/sd + 1/2/3/4/5
  • dx 五度組合 = dx/sd + 調値序列
  • dx 四角標調 = dx/sd + ping/shang/qu/ru
  • dx 阴阳四聲 = dx/sd + yin/yang

    sj ─╴時間日期╶─────

  • sj 日期 = sj + rq
  • sj 時刻 = sj + sk
  • sj 月份 = sj + yf
  • sj 曜日 = sj + yr

    bg ─╴八卦爻符╶─────

  • bg i:陽爻 o:陰爻 由下至上

    txj ─╴太玄經爻╶─────

  • txj t:天 d:地 r:人 由上至下

    mw ─╴盲文點字╶─────

  • mw 符點從上到下、左到右編號：
  • mw 1,2,3(,7),4,5,6(,8)

    yb ─╴國際音標╶─────

  • yb （尚未加入字符編碼）

    pz ─╴漢字批註╶─────

  • pz 日本 KanBun 研究 = pz + rw

    sb ─╴上標符號╶─────

  • sb 數字 = sb + 數字
  • sb 算符 = sb + j/d/k
  • sb 拉丁字母 = sb + ld
  • sb 希臘字母 = sb + xl

    xb ─╴下標符號╶─────

  • xb 數字 = xb + 數字
  • xb 算符 = xb + j/d/k
  • xb 拉丁字母 = xb + ld
  • xb 希臘字母 = xb + xl

    fk ─╴組合方塊╶─────

  • fk 橫向 = fk + h
  • fk 豎向 = fk + s
  • fk 明暗 = fk + m
  • fk 四角 = fk + j

    jt ─╴指向箭頭╶─────

  • jt 上/下/左/右 = jt + s/x/z/y
  • jt 左上/左下 = jt + zs/zx
  • jt 右上/右下 = jt + ys/yx
  • jt 左右/上下 = jt + zy/sx/yz/xs
  • jt 交叉對角 = jt + jc/dj
  • jt 順時針 = jt + ssz
  • jt 逆時針 = jt + nsz
  • jt 拐彎 = jt+s/x/z/y+g / jt+g

    tx ─╴幾何圖形╶─────

  • tx 正方形 = tx + fx/zfx
  • tx 三角形 = tx + sjx
  • tx 四邊形 = tx + sbx
  • tx 五邊形 = tx + wbx/wjx
  • tx 六邊形 = tx + lbx/ljx
  • tx 八邊形 = tx + bbx/bjx
  • tx 圓形 = tx + yx
  • tx 橢圓 = tx + ty
  • tx 半圓 = tx + by
  • tx 圓弧 = tx + yh
  • tx 菱形 = tx + lx
  • tx 星形 = tx + xx

    zb ─╴製表符號╶─────

  • zb 中間 = zb + zj
  • zb 橫/豎/斜/彎 = zb + h/s/x/w
  • zb 上/下/左/右 = zb + ss/xx/zz/yy

    bz ─╴信號標誌╶─────

  • bz 標準 = bz + bz
  • bz 地圖 = bz + dt
  • bz 環保 = bz + hb
  • bz 警告 = bz + jg
  • bz 交通 = bz + jt
  • bz 群體 = bz + qt
  • bz 體育 = bz + ty
  • bz 宗派 = bz + zp

    xz ─╴黃道星宮╶─────

  • xz 按排序 = xz + 數字
  • xz 按名稱 = xz + 簡拼/英文名

    tt ─╴天體符號╶─────

  • tt 按排序 = tt + 數字
  • tt 按名稱 = tt + 簡拼/英文名

    qp ─╴棋牌賭博╶─────

  • qp 骰子 = tz / qp+tz
  • qp 麻將 = mj / qp+mj
  • qp 象棋 = xq / qp+xq
  • qp 撲克牌 = pk / qp+pk
  • qp 多米諾骨牌 = gp / qp+gp
  • qp 遊戲塔羅牌 = tl / qp+tl

    pk ─╴撲克牌╶──────

  • pk 黑桃 Spades = pk + s
  • pk 紅桃 Hearts = pk + h
  • pk 方塊 Diamonds = pk + d
  • pk 梅花 Clubs = pk + c

    tl ─╴塔羅牌╶──────

  • tl 該牌組（Tarot Nouveau）用於塔羅牌遊戲

    dq ─╴國家地區╶─────

  • dq 採用 ISO 3166-1 國家地區代碼
  • dq dq + 二字母/三字母/三數字/英文名
